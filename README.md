## About The Project

**Alexandria Guide** is a fully functional tourism website that allows visitors to:
- Explore top attractions in Alexandria, Egypt
- Create personalized tour itineraries
- Calculate trip costs in real-time
- Save and manage their tour plans

Built as a complete **PHP/MySQL** web application with a responsive frontend, this project demonstrates full-stack development capabilities including user authentication, database integration, and dynamic content management.

---

##  Features

### 👤 User System
- **Sign Up / Sign In** - Secure user registration and login
- **Password Hashing** - Passwords are encrypted for security
- **Session Management** - Persistent user sessions with localStorage

### Tour Planner
- **Date Selection** - Choose your travel date
- **Traveler Counter** - Adjust number of travelers with +/- buttons
- **Attractions Selection** - Checkbox selection with real-time pricing
- **Hotel Selection** - Multiple accommodation options (Budget, Standard, Luxury)
- **Transportation Choice** - Bus, Taxi, or Private Driver
- **Live Cost Calculation** - Automatic total cost updates as you select
- **Save Tours** - Save itineraries to database

###  Gallery
- Responsive image grid
- Hover effects with overlay text
- High-quality images of Alexandria attractions

###  Reviews Section
- User testimonials
- Rating summary with visual bars
- Clean card-based design

### 📞 Contact Form
- User message submission
- Database storage of inquiries
- Professional FAQ section
